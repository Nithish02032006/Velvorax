Lead Capture Integration

Connect your website forms to Velvorax CRM

Web-to-Lead API

Use the following endpoint to submit leads from any external form (like Zoho Forms, Webflow, or custom HTML). No authentication is required for this specific endpoint, allowing direct public submissions.

Sample HTML Form 
<form action="[YOUR_CAPTURE_URL]" method="POST">
  <!-- Basic Info -->
  <input type="text" name="name" placeholder="Full Name" required>
  <input type="email" name="email" placeholder="Email Address" required>
  <input type="text" name="phone" placeholder="Phone Number">

  <!-- Company Details -->
  <input type="text" name="company" placeholder="Company Name">
  <input type="text" name="businessType" placeholder="Business Type (e.g. Private)">

  <!-- Lead Details -->
  <input type="text" name="address" placeholder="Address">
  <input type="number" name="budget" placeholder="Budget Amount">

  <select name="priority">
    <option value="Hot">Hot</option>
    <option value="Warm" selected>Warm</option>
    <option value="Cold">Cold</option>
  </select>

  <textarea name="requirement" placeholder="Requirement Details"></textarea>

  <!-- Optional: Redirect after submission -->
  <input type="hidden" name="redirect" value="https://yourwebsite.com/thank-you">

  <button type="submit">Submit</button>
</form>
How it works
  • Submit data via POST request.
  • Fields supported: name, email, phone, company, message.
  • New leads appear instantly in the Leads module.
  • Source will be automatically set to "Web Form".
Zapier / Make Integration

You can also use this URL as a Webhook destination in Zapier or Make.com to sync leads from 1000+ other apps.
